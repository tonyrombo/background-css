Algunas propiedades de Background en CSS

background position

ejemplo: 
				div{
					background-image: url("img/bird.jpg");
					background-repeat: no-repeat;
					background-position: center;
				}

background size

ejemplo: 
				div{
				    background: url(canary.jpg);
				    background-repeat: no-repeat;
				    background-size: 300px 250px;
				}

multiples backgrounds

ejemplo: 
                div{
				    background: url(img_tree.png), url(img_flwr.png);
                    background-size: 100% 100%;
                    background-repeat: no-repeat;
				}

multiples backgrounds

ejemplo: 
                div{
				    background: url(img_tree.png), 
                    background-size: 100% 100%;
                    background-repeat: no-repeat;
				}

				property values border-box,padding-box,content-box,initial,inherit
